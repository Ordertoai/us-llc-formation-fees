US LLC Formation Fees — Open Dataset

A primary-sourced, CC BY 4.0 dataset covering LLC formation costs across all 50 US states and the District of Columbia: state filing fee, annual report / franchise tax, and the Secretary of State source URL for each row.

Interactive tool

The data on this page powers a free interactive calculator. To estimate the total first-year and ongoing cost of forming an LLC in any state, use LLC Formation Cost Calculator, which applies this dataset plus filing-method and registered-agent options. State-by-state breakdowns are at LLC cost by state.
